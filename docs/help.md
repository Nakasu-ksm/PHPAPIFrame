# 助手函数
##### 本项目为方便开发提供了高效的助手函数

| 助手函数 | 描述 |
| ------------ | ------------ |
| json  | 以JSON格式返回（不结束程序）  |
| success  | 以JSON格式返回（结束程序）  |
| error  | 返回错误码和错误内容  |
| close  | 关闭API  |
| input  | 安全接受输入函数  |
| removexss  | 去除XSS  |
| db  | 数据库操作类  |
| no_empty  | 为空报错  |
| generate_key  | 生成一个key  |
